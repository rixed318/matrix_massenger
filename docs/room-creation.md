# Расширенные настройки создания комнат

Диалог «Create a new room» поддерживает расширенную конфигурацию комнат, чтобы сразу задать параметры приватности и модерации.

## Основные параметры

- **Room name** — обязательное поле. Используется для отображения в списках и заголовке чата.
- **Topic** — необязательное описание, показывается участникам в шапке комнаты.
- **Room alias** — локальный алиас без префикса `#` и домена. Позволяет поделиться человеко-понятным адресом (`#alias:server.tld`).
- **Mode** — выбор между:
  - `Chat` — обычная дискуссионная комната.
  - `Channel` — объявочный канал, где отправлять сообщения и реакции могут только модераторы (power level ≥ 50).
- **History visibility** — управляет тем, с какого момента новые участники видят переписку:
  - `Since invited` (`shared`)
  - `Since joining` (`joined`)
  - `Only while invited` (`invited`)
  - `Public archive` (`world_readable`)

## Приватность и федерация

- **Make this room public** — контролирует видимость в общедоступном каталоге.
- **Enable end-to-end encryption** — включает `m.megolm.v1.aes-sha2` для всех будущих сообщений.
- **Require an invite to join** — устанавливает `m.room.join_rules = invite`, блокируя вход без приглашения.
- **Disable federation** — записывает `m.federate = false` в `m.room.create`, ограничивая комнату только вашим homeserver'ом.

## Замедление чата

Включите переключатель **Enable slow mode**, чтобы задать минимальный интервал между сообщениями для участников (отправляется событие `org.matrix.msc3946.room.slow_mode`). Это помогает бороться со спамом и поддерживать размеренный темп обсуждений.

## Стартовый пост

Поле **Initial announcement** позволяет отправить приветственное сообщение сразу после успешного создания комнаты. Клиент отправляет обычное текстовое событие от вашего имени.

## Соответствие Matrix API

При сохранении настроек формируется запрос `createRoom` с параметрами `preset`, `initial_state`, `power_level_content_override` и `creation_content`. Это гарантирует, что выбранные опции доступны всем участникам сразу после появления комнаты.
