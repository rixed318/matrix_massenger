# Новые возможности мобильного приложения

## Общие компоненты
- Ввод сообщений использует общий пакет `@matrix-messenger/ui` с адаптацией под React Native, что обеспечивает единое поведение с веб-версией.
- Таймлайн комнат использует вспомогательные функции из `@matrix-messenger/core`, благодаря чему история сообщений синхронизирована между платформами.

## Сообщения и медиа
- Полноценный GIF/стикер-пикер с кэшированием через Async Storage и поддержкой избранного/истории.
- Менеджер черновиков и планировщик сообщений хранятся в Async Storage и восстанавливаются при повторном входе в комнату.
- Предпросмотр и планирование сообщений доступны через долгий тап на иконке часов в поле ввода.

## Secure Cloud и универсальный инбокс
- Secure Cloud срабатывания отображаются в универсальном инбоксе и дублируются нативными push-уведомлениями.
- Фильтры списка комнат позволяют быстро переключаться между непрочитанными, упоминаниями и предупреждениями Secure Cloud.

## Звонки
- Экран звонков интегрирован с платформенными CallKit/ConnectionService мостами (через `useNativeCallBridge`), обновляя состояние звонка на нативном уровне.

## E2E-тестирование
- Detox-конфигурация (`detox.config.ts`) настроена для Expo, доступны команды `e2e:build`/`e2e:test` для iOS и Android.
